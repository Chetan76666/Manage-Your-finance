# Budget Manager Application

A web application with Authentication and CRUD functionalities, 
to manage budget expenses.



## Developement Tools:
- React 16.x.x
- React Router
- React Router DOM
- Redux with redux-hooks
- Redux thunk
- Webpack 14.x.x
- Nodejs sever implementation


## Install dependencies
`npm install`

## Run Locally:
`npm run dev-start`

## Run on Node server:
`npm start`
